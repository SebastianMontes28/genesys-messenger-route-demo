Área de Nosotros (/nosotros)

Esta página simula la sección institucional de Kranon. Para garantizar una lectura sin interrupciones y evitar la superposición con elementos de la interfaz corporativa, el widget de Genesys debe permanecer oculto en esta ruta.

Evaluación de Ruta
/nosotros
Oculto (No Permitido)

Flujo de ejecución en esta ruta:

  1. 1
    Detección del Pathname

    El script app.js lee la propiedad de navegación del navegador e identifica la ruta como /nosotros.

  2. 2
    Comparación contra Lista de Permitidas

    El sistema verifica si la ruta coincide con las declaradas en el arreglo ALLOWED_MESSENGER_PATHS (que solo contiene la raíz /).

  3. 3
    Suscripción y Espera al SDK

    Al igual que en todas las páginas del sitio, se espera a que se dispare el evento Messenger.ready de Genesys para interactuar de forma segura con el SDK.

  4. 4
    Ocultación del Launcher

    Al confirmarse que la ruta no está permitida para el canal de chat, se ejecuta el comando nativo Genesys("command", "Launcher.hide", {}), asegurando que el widget desaparezca.

Recomendación CX

Control de Parpadeo Visual (Flickering)

Para una transición fluida, configure la visibilidad del launcher en Genesys Cloud Messenger como: "Hide until triggered by business logic".

Esta directiva le indica al widget que permanezca invisible hasta que la lógica de la página ordene mostrarlo explícitamente, evitando que el chat parpadee en rutas restringidas como /nosotros o /contacto. 

Genesys("subscribe", "Messenger.ready", () => {
  // Aplicar regla de ruta
  Genesys("command", "Launcher.show", {});
});
