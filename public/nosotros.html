Área de Nosotros (/nosotros)

Esta página simula la sección institucional de la compañía. Para garantizar una lectura sin interrupciones y mantener una presentación limpia de los contenidos, el chat flotante de Genesys permanecerá oculto.

Comportamiento en esta ruta

  1. I.
    Detección de Ruta

    El script app.js identifica que el usuario se encuentra en la sección /nosotros.

  2. II.
    Validación de Estado de Sesión

    El controlador verifica en sessionStorage si el widget estuvo activo previamente durante la sesión del usuario.

  3. III.
    Acción de Ocultación Inteligente

    Si el widget estuvo activo, se envían los comandos Launcher.hide y Messenger.close. Si el usuario ingresó de forma directa a esta ruta, **no se ejecuta ningún comando**, previniendo fallas y manteniendo el widget invisible por defecto.

Ajustes Recomendados

Control de parpadeo visual (Flicker)

En la consola de administración de Genesys Cloud, configure la visibilidad del launcher en: "Hide until triggered by business logic".

Esta directiva le indica al widget que permanezca invisible hasta que nuestro código ordene mostrarlo explícitamente, evitando que el chat parpadee en rutas restringidas. 

Genesys("subscribe", "Messenger.ready", () => {
  if (shouldShow) {
    Genesys("command", "Launcher.show", {});
  }
});
