Ruta: Contacto (/contacto)

Esta es la página de Contacto. En esta ruta, el launcher de chat debe estar oculto de manera predeterminada para evitar la saturación del canal o enfocar al usuario en otros medios.

¿Qué sucede aquí?

El script de control app.js intercepta la inicialización de Genesys mediante la suscripción a Messenger.ready.

  • La ruta actual se detecta y normaliza como /contacto.
  • Se compara con la lista de rutas autorizadas: ["/"].
  • Al no coincidir, se ejecuta: Genesys("command", "Launcher.hide", {});.
  • El botón flotante (launcher) de Genesys se oculta automáticamente.

Recomendación Clave

Configuración del Launcher

En la consola de Genesys Cloud Messenger, configure:
Launcher Visibility = "Hide until triggered by business logic".

Esto evita que el launcher nativo aparezca por defecto durante la carga de la página, eliminando el flickering o parpadeo visual en rutas restringidas como /nosotros o /contacto.

Lógica Principal Utilizada

window.Genesys("subscribe", "Messenger.ready", () => { if (shouldShow) { window.Genesys("command", "Launcher.show", {}); } else { window.Genesys("command", "Launcher.hide", {}); } });
